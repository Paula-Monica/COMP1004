This is a SPA made to help students with Timetabling.
You can create tasks by filling out the task details, this generates tasks segments.
You can select your availability in the timetable by pressing on a cell, which will then turn green.
These task segments can be dragged and dropped into the available timetable cells.
Tasks can be categoriesed using the key, where you can add the categories you want and assign them to the tasks.
